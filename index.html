Global migration flows and HIV-1 variant distributions

Below is a series of global maps displaying migration flows per 100,000 of the destination country population with HIV-1 variant distributions overlaid. Maps were created in R using the rnaturalearth package. Bilateral migration flow data is sourced from Abel and Cohen's international migration flow estimates (here). HIV-1 variant distribution data was compiled from a global molecular epidemiology database based on a systematic review and global survey. Created by Grace Oswald.
